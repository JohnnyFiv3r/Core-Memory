# Core Memory Agent Guide

The runtime canonical source for the MCP agent guide is packaged at:

`core_memory/integrations/mcp/core-memory-agent-guide.md`

It is packaged with `core-memory[mcp]` so the MCP server can expose it as the
`core-memory.agent-guide` prompt after installation. Keep this docs page as a
pointer; edit the packaged guide when changing the prompt/tool-description source
of truth.
